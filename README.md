# ObraLog Monorepo

This is a monorepo for ObraLog, built with TurboRepo, Node.js, and TypeScript.

## Structure

- **apps/web**: A Next.js web application.
- **packages/ui**: A shared React UI component library.
- **packages/tsconfig**: Shared TypeScript configurations.

## Getting Started

1.  Install dependencies:

    ```sh
    pnpm install
    ```

2.  Run the development server:

    ```sh
    pnpm dev
    ```

3.  Build the project:
    ```sh
    pnpm build
    ```
